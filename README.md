 # Animated To-Do List

A simple, animated to-do list web app built with HTML, CSS, and JavaScript.

## Features

- Add tasks to your to-do list
- Remove tasks with a smooth animation by clicking on them
- Modern, animated background and stylish UI

## Usage

1. Clone or download this repository.
2. Open `index.html` in your browser.
3. Type a task in the input box and click **Add**.
4. Click on a task to remove it with an animation.

## Files

- [`index.html`](index.html): Main HTML file
- [`style.css`](style.css): Styles and animations
- [`script.js`](script.js): JavaScript logic for adding/removing tasks
